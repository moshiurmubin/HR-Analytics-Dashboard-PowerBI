# HR-Analytics-Dashboard-PowerBI
Power BI dashboard analyzing HR attrition with key insights and interactive visuals
This project presents an HR Analytics Dashboard built using Power BI to analyze employee attrition and identify key business insights.

🚀 Overview
The dashboard was created to explore patterns in employee turnover and support data-driven decision-making.

🔧 Workflow
-Collected dataset from Kaggle
-Cleaned and structured the data for analysis
-Created calculated measures using DAX
-Built an interactive dashboard focusing on attrition trends

📊 Key Insights
-Higher attrition among early-career employees (age 26–35)
-Sales roles show higher turnover
-Employees in lower salary ranges are more likely to leave

🎯 Tools Used
-Power BI
-DAX
-Data Cleaning & Transformation

📌 Outcome
This dashboard highlights how combining data preparation, analysis, and visualization can uncover meaningful business insights.
